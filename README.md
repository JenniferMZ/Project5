# Project5
Traffic Lights

This program lights LEDs representing traffic lights flowing from East and West. It cycles through green, yellow, and red from one direction while lighting red from the other direction, then switches directions. It also responds to a button push, ensuring that lights for both directions are red before lighting a pedestrian light. The loop resumes after pedestrian flow is complete.

westButton (for pedestrian crossing) must be held to ellicit a response from the program.
TrafficLights_Final is the program submitted for grading. I tried to delete the extra ones.
